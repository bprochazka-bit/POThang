Placeholders

Templates can use these placeholders inside cells — they are substituted when a PO is rendered:

  • Header fields: {{ '{{po_number}}' }}, {{ '{{vendor}}' }}, {{ '{{ship_to}}' }}, {{ '{{date}}' }}, {{ '{{total}}' }}, {{ '{{notes}}' }}, {{ '{{revision}}' }}
  • Items loop: bracket rows with {{ '{{#items}}' }} and {{ '{{/items}}' }} markers; the rows between them are repeated per PO line. Inside, use {{ '{{item.index}}' }}, {{ '{{item.name}}' }}, {{ '{{item.qty}}' }}, {{ '{{item.unit_cost}}' }}, {{ '{{item.line_total}}' }}, {{ '{{item.model}}' }}, {{ '{{item.vendor}}' }}, {{ '{{item.vendor_sku}}' }}, {{ '{{item.url}}' }}, {{ '{{item.notes}}' }}
  • Conditional sections (single cell): {{ '{{#if var}}' }} … {{ '{{else}}' }} … {{ '{{/if}}' }}

Excel formulas (=SUM(F9:F11), etc.) and merged cells are preserved through the editor and render pipeline. The editor preserves basic styling (bold, alignment, background fill, number format); more exotic xlsx features (charts, conditional formatting, complex borders) are dropped on edit.
